Affichage de tous les élements d'une table. La fonction vérifie d'abord si la table est présente dans la table de gestion (gestion_tables).

Ensuite, elle vérifie s'il y a des attributs de cette table à afficher lors de l'appel à la liste (infos présentes dans la table gestion_crud_tables). Enfin, si on trouve au moins un attribut, on affiche les données.
